## 🚀 Getting Started

To use this project, you’ll need your own Ship24 API key.

### 1️⃣ Create a Ship24 Account  
Go to:  
👉 https://dashboard.ship24.com/login  

Sign up (or log in) and generate your API key.

### 2️⃣ Configure the Project  

Create a `config.js` file based on the provided example: `config.js.exemple`


Then open `config.js` and insert:

- Your **Ship24 API key**
- Your **tracking number**

### 3️⃣ Run the Project  

Once your API key and tracking number are set, you're ready to test the app.
